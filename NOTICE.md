# papercups75

**papercups75** is a maintained fork of [Papercups](https://github.com/papercups-io/papercups),
an open source live customer support tool.

The upstream project is in maintenance mode and no longer receives major updates.
This fork exists to keep the application current — upgrading dependencies and the
runtime stack, and continuing development.
